# Firebase CRUD Example

Full firebase application using DDD and GetX.
Effective dart

## Run the app

Flutter Version: `1.22.6 stable channel`

    cd firebase_crud_example/
    flutter create .

Configure `Firebase` into your `Android/IOS` folder

    flutter run 

No extra steps required.
